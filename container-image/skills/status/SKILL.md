# Status Report

When the user asks about your status, health, or how you're doing (in a technical context), provide a system status report.

## How to Respond

Use the `get_memory_stats` MCP tool to gather memory statistics, then report:

1. **Memory Status**: How many semantic memories, meta memories, and external documents are stored
2. **Session Info**: Whether this is a resumed session or a fresh start
3. **Available Tools**: Confirm which MCP tools and capabilities are accessible
4. **Identity**: Briefly confirm your configured identity/role

Keep the report concise and factual. Format as a short bulleted list.
